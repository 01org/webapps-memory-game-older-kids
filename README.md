# INTRODUCTION
Memory Game for Older Kids is a game where player tries to find pair of cards which have same image.

It has been tested on the following browsers/platforms:
* Tizen/WRT
* Tizen/Crosswalk
* Android/Crosswalk
* Tizen IDE
* Chrome/file
* Chrome/extension
